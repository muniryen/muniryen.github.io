Sürüm 0.0.1 ve yukarısı

Version 0.0.1 and above

12.07.2025

  1. Uygulama, herhangi bir kişisel verinin işlenmesini, paylaşılmasını veya barındırılmasını gerektirmeyecek şekilde tasarlanmıştır. Dolayısıyla geliştirici tarafından kişisel veriler işlenmemekte, paylaşılmamakta veya barındırılmamaktadır.

    The application is designed in such a way that it does not require any personal data to be processed, shared or hosted. Therefore, personal data is not processed, shared or hosted by the developer.

  2. Google Mobile Ads SDK, reklam, analiz ve sahtekarlık önleme amacıyla IP adresini, Android reklam kimliğini, uygulama seti kimliğini vb. otomatik olarak toplar ve paylaşır. Daha fazla bilgi için lütfen bkz:

    The Google Mobile Ads SDK collects and shares IP address, Android advertising (ad) ID, app set ID etc. automatically for advertising, analytics, and fraud prevention purposes. For more information please see:

    Google Play's data disclosure requirements | Android

    How Google uses information from sites or apps that use our services

  3. Uygulama hesap oluşturmadan, oturum açma işlemi gerektirmeden kullanılabilmektedir. Ne bu gibi işlemler için gerekli bilgiler toplanır ne de paylaşılır.

    The application can be used without creating an account or requiring a login. Information required for such operations is neither collected nor shared.

  4. GPS hizmeti çalıştırılmaz ve konum bilgisi kullanıcının izni olmadan paylaşılmaz. Kullanıcı konum bilgisine ihtiyaç yoktur. flutter_map eklentisi sadece ülkelerin konumunu harita üzerinde göstermek için kullanılır. flutter_map widget'ında, kodlar kullanıcıdan mevcut konum bilgilerini istemeyecek şekilde ayarlanmıştır.

    GPS service is not run and location information is not shared without the user's permission. There is no need for user location information. flutter_map plugin is only used to show the location of countries on the map. In flutter_map widget, the codes are set to not prompt the user for current location information.

  5. Veri setlerinin kaynağı https://acikveri.bizizmir.com/ adresidir.

    Dataset source is https://acikveri.bizizmir.com/ .
